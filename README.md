Lambda Quest
=================

Live-coding Scheme environment in the browser

Try it: https://lambda-quest.pages.dev

### Features
* Gambit Scheme
* Live evaluation of forms in the editor
* Asynchronous Canvas rendering
* Web Audio support
* Animations and timeouts
* Autocomplete in the editor
* REPL
